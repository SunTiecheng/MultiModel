输入视频放在video文件夹下

1.抽帧
python video_process.py -i (输入路径) -o (输出路径) -t (目标帧数)
python video_process.py -i ./video/flower2.mp4 -o ./frame -t 150

2.运行2dgs自动化脚本run_2dgs.sh

./run_2dgs.sh

3.运行文件检查file.py

python file.py copy -n (物体名)
python file.py copy -n flower2

4.运行膨胀mask
python dilate.py -s (膨胀内核大小，默认为10)
python dilate.py -s 50

5.运行去除背景
python run.py -i (物体名).ply
python run.py -i flower2.ply


配置环境：
conda env create --file 2d-gaussian-splatting/environment.yml
pip install rembg[gpu,cli]
pip install -r xmem/requirements.txt
